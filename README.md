# Phigros 第九章预热网站

一个用于 Phigros 第九章预热活动的静态站点，包含倒计时、解密入口及事件时间轴。

## 页面说明

- **index.html** — 预热首页  
  包含顶部品牌标识、主标题、导航按钮，以及三个倒计时模块（PMT、预热爆料、预期更新时间）。

- **arg.html** — 解密进度页  
  提供三个功能入口：关注 B 站账号、查看事件时间轴、加入 QQ 群。

- **time.html** — 事件时间轴  
  按时间顺序梳理第九章预热过程中的关键事件与线索。

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需任何第三方库或框架
- 适配移动端与桌面端
